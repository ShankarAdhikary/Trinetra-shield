# TRINETRA Chrome Extension

## Overview
TRINETRA is a powerful Chrome extension designed to enhance your browsing experience with advanced features tailored for productivity and security.

## Features
- **Enhanced Security**: Protects against phishing attacks and malicious websites.
- **Productivity Tools**: Includes tools for task management and time tracking.
- **Custom Notifications**: Get alerts based on your browsing habits and preferences.

## Setup Instructions
1. **Install Chrome**: Ensure you have the latest version of Google Chrome.
2. **Download the Extension**: Visit the Chrome Web Store and search for "TRINETRA".
3. **Add to Chrome**: Click on "Add to Chrome" and confirm the installation.
4. **Setup Account**: Create an account or log in if you already have one.
5. **Configuration**: Go to settings to customize features according to your needs.

## Backend Integration Guide
- **APIs Used**: The extension connects to our backend using REST APIs.
- **Authentication**: Utilize OAuth2 for user authentication.
- **Data Storage**: User preferences and history are stored securely in a cloud database.
- **Error Handling**: Implement error handling for API responses to enhance user experience.

## Troubleshooting
- **Extension Not Working**: Ensure you are using the latest version of Chrome.
- **Unable to Log In**: Check your internet connection and verify your credentials.
- **Feature Malfunctions**: Restart the extension or reinstall it from the Chrome Web Store.

## Development Guide
- **Codebase Overview**: The extension is built using JavaScript, HTML, and CSS.
- **Setting Up a Local Development Environment**:
  1. Clone the repository: `git clone https://github.com/ShankarAdhikary/Trinetra-shield.git`
  2. Navigate to the directory: `cd Trinetra-shield`
  3. Open Chrome and go to `chrome://extensions/`
  4. Enable "Developer mode" and click on "Load unpacked"
  5. Select the extension directory to test locally.
- **Testing**: Use Chrome's built-in developer tools for debugging.
